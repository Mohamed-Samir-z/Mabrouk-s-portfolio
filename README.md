🚀 Web Tech Lab Project - Portfolio for Mohamed Mabrouk📝 Project OverviewThis is a multi-page professional portfolio website designed and developed for a colleague, Mohamed Mabrouk (Student at Faculty of Computers and AI, Al-Azhar University). The project was created as part of the Internet Technologies Lab Exam to demonstrate proficiency in core web technologies.  🛠️ Key FeaturesMulti-page Architecture: Includes Home, About, Tech & Innovation, and Contact pages.  Custom Form Validation: A robust JavaScript system to validate university emails (@azhar.adu.eg) and secure password lengths.  Semantic HTML5: Usage of headings, lists (ordered & unordered), tables, and forms.  Responsive Styling: Implementation of External, Internal, and Inline CSS to achieve a modern "Dark & Green" engineering theme.  UX/UI Focus: Clean layout, easy navigation, and consistent design across all pages.  💻 Technologies UsedHTML5   CSS3 (Flexbox, Root Variables, Custom Hover Effects)   JavaScript (DOM Manipulation & Form Validation)
